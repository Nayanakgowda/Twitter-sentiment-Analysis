# Twitter-sentiment-Analysis
Twitter Sentiment Analysis project using Python, Machine Learning, and NLP to classify tweets into positive, negative, and neutral sentiments.

## Objectives
- Collect and analyze Twitter dataset
- Clean and preprocess text data
- Perform exploratory data analysis (EDA)
- Build and train a sentiment classification model
- Evaluate model performance

## Tools and Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Natural Language Processing (NLP)

## Project Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis
4. Feature Extraction
5. Model Training
6. Model Evaluation
